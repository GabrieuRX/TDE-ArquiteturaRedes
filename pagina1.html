PROTOCOLOS TD3-REDES

IP
O IP define como pacotes de dados são endereçados e roteados entre dispositivos.
TCP
TCP opera na camada de transporte e garante a entrega confiável de dados entre sistemas.
UDP
também opera na camada de transporte mas não garante a entrega confiável de dados entre sistemas.
SMTP
SMTP é o protocolo padrão para envio de e-mails na internet. Ele define as regras de transmissão.
DNS
É responsável por traduzir nomes de domínio legíveis por humanos em endereços IP
IMAP
O IMAP é um protocolo utilizado para acessar e gerenciar e-mails armazenados em servidores.
HTTP
HTTP é o protocolo utilizado para a transferência de documentos de hipertexto na (World Wide Web).
P2P
modelo descentralizado para compartilhamento direto de arquivos entre usuários, sem depender de servidores centrais.
